# Security Policy

## Supported Versions

Wall Planner is currently in active development. Security fixes are applied to the latest `main` branch.

## Reporting a Vulnerability

Please **do not** report security vulnerabilities in public GitHub issues.

Instead, report privately by emailing:

**lee@pasifull.co.uk**

Please include:

- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if known)

We will acknowledge receipt as quickly as possible and aim to provide an initial response within 72 hours.
